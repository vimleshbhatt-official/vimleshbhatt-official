## Design Principles

**01 — Observability first**
A system you can't measure is a system you can't trust. APM, tracing, and structured logs are not afterthoughts.

**02 — AI in the critical path**
LLMs belong in production, not playgrounds. Latency budgets, fallback paths, and token costs are engineering constraints like any other.

**03 — Data integrity over speed**
PostgreSQL is not just storage. Indexing, MVCC, and query planning are architectural decisions.

**04 — APIs as contracts**
Every endpoint is a promise. FastAPI + Pydantic makes those promises explicit and enforced.
